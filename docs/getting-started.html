System Overview

This document provides setup and configuration instructions for the bsuppla static analysis framework.

Table of Contents

  • Prerequisites
  • Build Process
  • Configuration
  • Environment Variables

1. Prerequisites

The following system requirements must be satisfied before building bsuppla:

Component Version Purpose
Rust Toolchain 1.70+ (stable) Compilation and execution
Docker Latest Image acquisition
Operating System Linux (primary) Execution environment

2. Build Process

The system is built from source using Cargo, the Rust package manager.

2.1 Repository Clone

git clone https://github.com/gabrielchboff/bsuppla.git
cd bsuppla

2.2 Compilation

cd bsuppla/bsuppla
cargo build --release

2.3 Output Binary

Following successful compilation, the executable is located at:

bsuppla/bsuppla/target/release/bsuppla

2.4 Installation

# System-wide installation (requires privileges)
sudo cp target/release/bsuppla /usr/local/bin/

# Or use the provided installer
bash scripts/install.sh

3. Configuration

bsuppla operates with minimal configuration. All parameters are passed via command-line arguments or environment variables.

3.1 Command-Line Interface

bsuppla <IMAGE> <OUTPUT_TAR> [ALLOWLIST] [BASELINE]

Arguments:
  IMAGE           Docker image name (e.g., alpine:latest)
  OUTPUT_TAR      Path for docker save output
  ALLOWLIST       Optional allowlist file
  BASELINE        Optional baseline file for comparison

4. Environment Variables

Runtime behavior can be modified through environment variables:

Variable Values Function
BSUPPLA_SKIP_DOCKER 1, true Skip docker pull/save operations
BSUPPLA_SKIP_SCAN 1, true Skip analysis stage
BSUPPLA_OUTPUT_DIR path Extraction directory (default: container_fs)

4.1 Usage Examples

Standard Analysis

bsuppla alpine:latest alpine.tar

With Allowlist

bsuppla alpine:latest alpine.tar allowlist.txt

Baseline Comparison

bsuppla alpine:latest alpine.tar allowlist.txt baseline.txt

Skip Docker (existing tar)

BSUPPLA_SKIP_DOCKER=1 bsuppla alpine:latest existing.tar

5. Verification

Verify installation:

bsuppla --version
# Output: bsuppla x.y.z
